How to Run the Project
1. Prerequisites
Ensure you have Python 3.8+ installed on your system. You can check your version by running:

python --version

2. Install Required Libraries
This project relies on four core data science libraries. You can install them all at once using pip:


pip install matplotlib seaborn pandas numpy

3. Execution
Download the dashboard.py file.
Open your terminal or command prompt.
Navigate to the folder containing the file.

Run the script:
python dashboard.py
